SPA(-PRO) Mail Server (2003/@Solomon) から E-Post Mail Server へ移行する場合
SPA(-PRO) Mail Server (2003/@Solomon) から E-Post Mail Server への移行手順は、旧製品の各サービス停止後、次の通り行ってください。

1.SPA(-PRO) Mail Server (2003/@Solomon) の「バージョン情報」タブ画面より各サービスを停止させます。

2.E-Post Mail Serverのインストールを開始。

3.インストール完了後に、E-Post Mail Server（赤いポストマーク）アイコンを起動します。設定ウイザードが起動しますが、キャンセルボタンを押します。

4.既存の設定が引き継がれた状態で、E-Post Mail Serverの各サービスのインストールの確認ダイアログが表示されるので、「OK」をクリックし設定します。

5.設定画面であるMail Controlが表示されるので、設定が正しく引き継がれたか念のため確認してください。

6.設定が正しければ、「サービス制御」タブ画面にて各サービスを開始します。
