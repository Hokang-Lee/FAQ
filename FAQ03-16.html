Windows Vistaへの導入時の注意

Windows Vista では、Administratorアカウントについても、User Account Control（UAC）が有効となっていると、Administratorアカウントでもサービス登録や操作ができませんので、UACの設定を解除してから、インストールやプログラムの起動や操作を行う必要があります。

■Windows Vistaに導入する前の操作　[UACの解除]
　Administratorアカウントでログイン
　　コントロールパネル
　　　→　ユーザアカウント
　　　　→　ユーザーアカウント制限の有効化または無効化
　　　　　→　[ユーザーアカウント制御(UAC)を使ってコンピュータの
　　　　　　　保護に役立たせる]チェックボックスをオフにする
　設定後には必ずシステムを再起動してください。

■インストールとファイヤウォールなどの設定
　再起動後には以下の操作を行います。
  1. Administratorアカウントでログイン
  2. E-Post Mail Serverシリーズをインストール
  3. Windowsファイヤウォール設定でSMTP/POP3/IMAP4ポートの許可を追加
  4. 各サービス（SMTP/POP3/IMAP4）を開始
なお、Windows Vista Home Basic、Windows Vista Home Premium は、ドメインに参加できない仕様のため、当然の結果として、Active Directory（AD）によるアカウント連携での運用はできませんので注意してください。
