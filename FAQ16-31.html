BossCheck Server 上長承認機能を全体で抑止させ使わない方法について new!

BossCheck Server 上長承認機能を全体で抑止させ使わない方法ですが、簡単にできます。
E-Post BossCheck Option (x64) の画面（小さなダイアログボックス）を呼び出して、「上長承認を有効にする」チェックボックスをオフにします。その後、上記チェックボックスをオフにした後は、E-Post Mail Controlの「サービス制御」タブ画面よりEPSTRSサービスを停止→開始と、サービス再起動をすると、その後からは上長承認機能が無効になります。
上長承認の条件設定を記述している "sender.dat" についてはそのままでもかまいませんし、ていねいに１つずつ承認依頼を行う条件設定部分を削除してもかまいません。全体で上長承認機能が無効になると、"sender.dat" 内に記載している承認依頼を行うオプションを意味する動作コードの "1","2","3" といったコードがパスされることになります。
