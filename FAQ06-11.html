Apache 2.2.x でMobile Mailerを動作させるには

Mobile Mailerを Apache 2.2.x で動作させるには、下記の通り、httpd.confファイルにエイリアスの設定に加え、関連する権限の設定を新たに追加することが必要になります。
下記のどちらかの方法で記述します。この記述により、Apache 2.2.6で動作することを弊社でも確認しています。

<IfModule alias_module>
　：
　# エイリアスを追加
　Alias /webmail/ "c:/program files/epost/webmail/"　←追加
　：
</IfModule>

# <Directory> ディレクティブを追加
<Directory "c:/program files/epost/webmail">　←
　Allow from all　　　　　　　　　　　　　　　　　｜ ---- 追加
</Directory>　　　　　　　　　　　　　　　　　 ←
あるいは
<IfModule alias_module>
　　：
　# エイリアスを追加
　Alias /webmail/ "C:/Program Files/EPOST/webmail/"　←追加
　　：
</IfModule>

# <Directory> ディレクティブを追加
<Directory "C:/Program Files/EPOST/webmail">　←
　AllowOverride None　　　　　　　　　　　　　　　｜
　Options None　　　　　　　　　　　　　　　　　　｜ ---- 追加
　Order allow,deny　　　　　　　　　　　　　　　　 ｜
　Allow from all　　　　　　　　　　　　　　　　　　｜
</Directory>　　　　　　　　　　　　　　　　　　 ←
注：アクセス権の定義部分のフルパス末尾は、ディレクトリ記号のスラッシュ"/"を記述しません。
