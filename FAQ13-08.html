エイリアス機能はありますか？

E-Post Mail Server (x64) シリーズ、E-Post SMTP Server (x64) シリーズ製品は、エイリアス機能を持っています。エイリアスは実アカウントと関連づけ、基本的にエイリアスと実アドレスが一対一の関係で設定します。エイリアスどうしの関連づけは対応していません。また「エイリアス一 対 実アドレス多」のような設定はE-Postシリーズでは不可です。

なお、設定したエイリアスは必要ライセンスの対象としてカウントされます。実アドレス数とメーリングリスト名、そしてエイリアス数とを合わせた総合計が、ライセンス数におさまるように運用しなくてはなりません。そのため、仮に500人分の実アドレスとした場合、そのすべてにエイリアスを一対一で関連付け設定すると、必要なライセンス数は、倍の1000 User分が必要だということになります。
