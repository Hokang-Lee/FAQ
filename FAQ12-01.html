アップデートはどの程度の頻度で行われるのでしょうか？

E-Post Mail Server (x64) シリーズや E-Post SMTP Server (x64) シリーズ製品、E-Post Secure Handler (x64)、E-Post BossCheck Option (x64) や E-Post LGWAN Option (x64) などのアップデートについては、年に数回程度を予定しています。アップデートはユーザー登録を完了されたユーザー様にお送りするIDで「サポート２」にログインし、「差分アップデートプログラム」（アップデータ）をダウンロードして実施していただきます。次年度更新時に添付しているCDの中にも「差分アップデートプログラム」を収録しており、そちらをお使いいただいてもかまいません。
なお、定期的なアップデート以外に、セキュリティ上の問題や不具合などが新たに判明したときは、随時、実施される予定です。

32bit版 E-Postシリーズ、SPA-PRO Mail ＠Solomonシリーズなどの旧製品は、既にサポート終了しているため、今後はアップデートはされません。
