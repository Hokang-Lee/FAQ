BossCheck Server 承認依頼メールで quoted-printable でエンコードされた件名が文字化けや文字欠けする現象について

BossCheck Server 承認依頼メールで quoted-printable でエンコードされた件名が文字化けや文字欠けする現象が起きることがあります。これは、送信クライアント側の仕様や特性により、quoted-printable でエンコードされた件名部分のパックされた各行末尾に空白スペースが入れられているために、文字化けや文字欠けが発生するためです。
送信クライアント側の特殊な仕様により quoted-printable でエンコードされた件名部分の各行末尾に空白スペースが入れられている状況でも、E-Post側で対策を取り、対応する修正モジュールをリリースしています。20151225修正差分プログラム以降を適用することで、この問題を解決できます。適用後のEPSTRSのバージョンはv4.92になります。ちなみにそれ以降のバージョンでもこの対策は入っています。

[64bit版]
