Windows 2000への導入時の注意

1. Windows 2000 をインストールし、IISを組み込むとデフォルトで「簡易SMTPサービス」が自動開始になりますので、「簡易SMTPサービス」が”手動”でさらに”停止”されていることを確認した後、E-Postシリーズのサービスの起動を行って下さい。

2. Windows 2000 のユーザー権利設定は以下のとおり行います。
なお、ここでの設定は、アカウントマネージャが自動的に権利設定を行いますので、手動での設定は特に必要ありません。

Active Directory が未インストールのサーバの場合
　[メニュー]->
　　[プログラム]->
　　　[管理ツール]->
　　　　[ローカル・セキュリティ・ポリシー]->
　　　　　[ローカルポリシー]->
　　　　　　[ユーザー権利の割り当て]->
　　　　　　　[バッチジョブとしてのログオン]->

Active Directory がインストールされドメインコントローラとして機能している場合
　[メニュー]->
　　[プログラム]->
　　　[管理ツール]->
　　　　[ドメインコントローラ・セキュリティ・ポリシー]->
　　　　　[ローカルポリシー]->
　　　　　　[ユーザー権利の割り当て]->
　　　　　　　[バッチジョブとしてのログオン]->
追加ボタンを押してメールグループ（IMSUsers）を追加して下さい。
