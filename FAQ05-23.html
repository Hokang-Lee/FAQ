E-Post BossCheck Option (x64) で使用している設定ファイル＆レジストリ updated! 
■共有情報−[メール作業フォルダ]\regフォルダ内の設定ファイルで設定
    ※設定ファイルの拡張子種類は、次の通り値の型を表わす。レジストリのBINARY,DWORD,STRINGの型に該当。
       （ .0 = BINARY,  .1 = DWORD,  .2 = STRING及びその他の形式 ）
     ※秒数や回数、サイズなどのDWORD値は原則としてすべて10進数表記で設定。

[メール作業フォルダ]内
→REG
 →SOFTWARE
  →EMWAC
   →IMS
     →MailApprovalLog.1        (DWORD  Default 1) 上長承認履歴（記録ログ）の保存有無
                                             ( 1:保存する, 0:保存しない )
     →MailApprovalManager.2  (SZ   Default NULL) 上長承認メールシステム管理者
     →MailApprovalRes.1        (DWORD  Default 1) 保留通知メールの有無
                                             ( 1:保留通知あり, 0:保留通知なし )
     →MailApprovalDepath.1   (DWORD  Default 1) 代理承認者の代理設定可能な階層
                                             ( 1〜:回帰的に可能な代理承認の深さ ）
     →MailApprovalNotification.1  (DWORD  Default 1) 開封通知メールを添付ファイル条件除外
                                             ( 1:除外する, 0:除外しない )
     →MailApprovalNum.1      (DWORD  Default 2) 2人を超える承認者数 ( 例:承認者3人…3 )
         ただしフルパス表記[メールボックスフォルダ\作成ドメイン名\アカウント名]長さ合計が最大200バイトまで
         （例）承認者3人＝アカウントがttanaka、ssuzuki、ssatoのケース
           C:\mail\inbox\testdomain.jp\ttanaka … 35バイト
           C:\mail\inbox\testdomain.jp\ssuzuki … 35バイト
           C:\mail\inbox\testdomain.jp\ssato … 33バイト   計103バイト
     →MailApprovalDomain.2  (SZ   Default NULL) サブドメインをエイリアス設定したときの
                                            サブドメイン名 ( 例:sub )
     →MailApprovalLogLevel.1 (DWORD  Default 0) 履歴に送信元IPや送信先を含むか含まないかの設定
                                             ( 0:含まない, 1:含む ) (EPSTRS v4.82b〜) 
     →MailApprovalCodepage.1 (DWORD  Default 932) 承認依頼メールのmailtoリンクの生成時の文字コード設定
                                             (932(Shift-JIS), 65001(UTF-8),20932(JP_EUC),
                                              50220 or 50222(JP_JIS))(EPSTRS v4.86〜) 

■個別情報−レジストリに設定
HKEY_LOCAL_MACHINE
 →SOFTWARE
  →EPOST
   →BOSSCHECK
     →proxymessage40      (SZ) 代理承認時の通知メールメッセージ（JISエンコード文字列）
     →proxymessage41      (SZ) 代理承認解除時の通知メールメッセージ（JISエンコード文字列）

HKEY_LOCAL_MACHINE
 →SOFTWARE
  →EMWAC
   →IMS
     →MailApprovalHtml     (DWORD  Default 0) 上長承認依頼メールをHTML形式メールに変更するオプション
                                         ( 0:PLAIN 1:HTML ) (EPSTRS v4.A3〜)
     →MailApprovalHtmlColumn      (DWORD default 78) 承認依頼メールHTML形式で1行当たりの文字数を正規化するオプション
                                         (EPSTRS v4.AI〜)

HKEY_LOCAL_MACHINE
 →SYSTEM
  →CurrentControlSet
   →Services
    →EPSTRS
      →MailApprovalMessNum (DWORD  Default 100)
        [作成リトライ回数] 100(デフォルト), 0:リトライしない (EPSTRS v4.AH〜)
        BossCheck有効時の通知メール作成が失敗したときのリトライ回数
