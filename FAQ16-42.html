E-Post Secure Handler (x64) で使用している設定ファイル＆レジストリ updated! 
■共有情報−[メール作業フォルダ]\regフォルダ内の設定ファイルで設定
    ※設定ファイルの拡張子種類は、次の通り値の型を表わす。レジストリのBINARY,DWORD,STRINGの型に該当。
       （ .0 = BINARY,  .1 = DWORD,  .2 = STRING及びその他の形式 ）
     ※秒数や回数、サイズなどのDWORD値は原則としてすべて10進数表記で設定。

[メール作業フォルダ]内
->REG
 ->SOFTWARE
  ->EMWAC
   ->IMS
     ->MailApprovalReversType.1     (DWORD  Default 0)  0:送信しない, 1:送信元, 2:送信先指定
                                                  セキュアハンドラによる承認依頼メール送信エラー時の通知先
     ->MailApprovalReversFrom.2    (SZ  Default NULL) 上記2を指定した場合の送信先
     ->SHDLCease.1                    (DWORD  Default 1) 1:隔離する 0:隔離しない
                                                  自動判定でスパム判定されたメールの隔離有無の設定項目 (admiappr v1.17〜)
     ->SHDLHedearAdd.1             (DWORD  Default 0) 0:付加しない 1:付加する
                                                  自動判定でヘッダを付加するための項目 (admiappr v1.17〜)
     ->SHDLHedear.2                  (SZ  Default "X-SECUREHNDLER-Result")
                                                  自動判定で付加ヘッダの文字列指定項目 (admiappr v1.17〜)
     ->SHDLHSPAMTag.2              (SZ  Default NULL) 
                                                  自動判定でSubjectヘッダに付加する文字列指定項目 (admiappr v1.17〜)
