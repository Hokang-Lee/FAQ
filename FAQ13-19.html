ログ分析（解析）機能はありますか？

ログ分析（解析）機能を持つツールとして、E-Post Loganalyzer と呼ばれるユーティリティを E-Post Mail Server (x64) シリーズ、E-Post SMTP Server (x64) シリーズ各製品に付属しています。

解析した結果は、ブラウザで表示されるほか、解析した結果を決められた日時にメールで送信することも可能です。

解析可能なログの種類は以下の通りです。
  • 接続マシンログ → acceptlog フォルダ内ログの解析を行います。
  • SMTP受信ログ → inlog フォルダ内ログの解析を行います。
  • SMTP送信（外部ユーザー）ログ → outlog フォルダ内ログの解析を行います。
  • SMTP送信（内部ユーザー）ログ → outlocallog フォルダ内ログの解析を行います。
  • POP3受信者ログ → pop3log フォルダ内ログの解析を行います。
  • IMAP4受信者ログ → imap4log フォルダ内ログの解析を行います。
  • SMTP送信エラーログ → faillog フォルダ内ログの解析を行います。
  • メール・ウイルスフィルタログ → viruslog フォルダ内ログの解析を行います。
