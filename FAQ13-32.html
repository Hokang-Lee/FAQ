Windows Server 2003のMSCSで構成したHAクラスタ環境でメールサーバは動作しますか？

E-Post Mail Server シリーズ、E-Post SMTP Serverシリーズ製品は、Windows Server 2003 Enterprise Edition以上に付属しているMSCS（Microsoft Server Cluster Service）で構成したHAクラスタ環境での動作について、2007年5月1日の差分アップデートプログラム適用以降、正式に対応しました。なお、ダウンロードコーナーからダウンロードできる評価版プログラムについては、これらはすべて反映された最新バージョンとなっています。

MSCSで構成したHAクラスタ環境でメールサーバを動作させるには、下記を参照してください。

・FAQ　「MSCSクラスタ環境でメールサーバを設定する注意点」
・テクニカルホワイトペーパー『MSCS方式クラスタリングメールサーバ構築ガイド』 （PDF形式）

なお、マイクロソフトのWindows Server 2003サポート終了告知に合わせて、E-Post Mail Serverシリーズ、2016年5月の時点でE-Post SMTP Serverシリーズ製品でもWindows Server 2003でのサポート対応を終わらせていただきました。
