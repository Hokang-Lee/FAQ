アンインストール時に完全削除する方法

評価版・正規版にかかわらず、32bit版のE-Post Mail Server / E-Post SMTP Server シリーズ、64bit版のE-Post Mail Server (x64)/ E-Post SMTP Server (x64) シリーズ製品の完全なアンインストールを行いたいとき、E-Post関係のプログラムや設定情報など完全削除を行うには、次の手順と方法を取ってください。
  1. 事前にE-Post Mail Control画面からE-Post関係の各サービスを[停止]ボタンクリックで停止。E-Postクラスタ方式でのクラスタ構成の場合はIPWatcherサービスも停止。
  2. 同上画面で各サービスを[削除]ボタンクリックで削除。
    　※サービスを削除すると、ライセンスキー登録情報も合わせてクリアされます。従って、サービスを[削除]ボタンで削除するときは、その後のことも十分にご考慮の上、実行してください。
  3. 「プログラムの追加と削除」からアンインストールを実行。
    　※1.の手順を忘れ、サービスを停止しないでアンインストールを実行すると、サービスだけがアンインストールされずに残ってしまいます。
  4. プログラムインストールフォルダの削除
    　［E-Post(x64)(64bit版)の場合の既定値］
    　C:\Program Files\EPOST\MS および C:\Program Files\EPOST
    　［E-Post 32bit版の場合で32bitOS環境の既定値］
    　C:\Program Files\EPOST\MS および C:\Program Files\EPOST
    　［E-Post 32bit版の場合で64bitOS環境の既定値］
    　C:\Program Files (x86)\EPOST\MS および C:\Program Files (x86)\EPOST
  5. メールボックスフォルダの削除
    　［既定値での例］ C:\mail\inbox\%USERNAME%
  6. メール作業関連フォルダの削除
    　［既定値での例］ C:\mail
  7. レジストリの削除
    　※アンインストール後に下記のレジストリキーが残っている場合は削除。EMWAC自体およびEPOST自体を削除します。ちなみにEPOST自体を削除を削除すれば、その下のEPOST\IMSも削除されます。
    　［E-Post(x64)(64bit版)の場合の既定値］
    　HKEY_LOCAL_MACHINE\SOFTWARE\EMWAC
    　HKEY_LOCAL_MACHINE\SOFTWARE\EPOST
    　HKEY_LOCAL_MACHINE\SOFTWARE\EPOST\IMS
    　［E-Post 32bit版の場合で32bitOS環境の既定値］
    　HKEY_LOCAL_MACHINE\SOFTWARE\EMWAC
    　HKEY_LOCAL_MACHINE\SOFTWARE\EPOST
    　HKEY_LOCAL_MACHINE\SOFTWARE\EPOST\IMS
    　［E-Post 32bit版の場合で64bitOS環境の既定値］
    　HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node\EMWAC
    　HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node\EPOST
    　HKEY_LOCAL_MACHINE\SOFTWARE\Wow6432Node\EPOST\IMS

    　※EPOSTとEPOST\IMSを分けて書いたのには少々意味があります。E-PostシリーズのオプションシリーズのソフトウェアがEPOSTの下にIMSとは別にキーがそれぞれおかれる構造になっており、EPOST\IMSはメールサーバ本体が利用するため、ここでは分けて明記しています。レジストリ全体の詳細については、下記を参照してください。
