SMTPローカル送信ログ（outlocallog）について

SMTPローカル送信ログ（outlocallog）は、内部ドメインへの配送を実行し、メールボックスへ保管した記録です。書式はoutlogと同一です。
内部ドメイン宛への送信記録であるSMTPローカル送信ログ（outlocallog）は、E-Post Mail Control画面の［ログ設定］タブ内に設定する項目はありませんが、レジストリを設定することで取得可能です。
記録を採る場合は、以下の通り、レジストリを操作してDWORD値10進で作成、値を"1"に設定した後、SMTP配送サービス(EPSTDS)を再起動してください。

HKEY_LOCAL_MACHINE
　SYSTEM
　　->CurrentControlSet
　　　->Services
　　　　->EPSTDS
　　　　　->MailSendLocallogEnabled
　　　　　　　　(DWORD) Default 0:ログ記録しない、1:ログ記録する

 記録ログは、E-Post Mail Server / SMTP Server シリーズの場合、[メール作業フォルダ￥outlocallog￥コンピュータ名] フォルダ下に日付単位で作成されます。

SMTPローカル送信ログ（outlocallog）の内容は、以下の通りです。
<メッセージID＠マシン名>　<sent on [記録日時]>　<to 宛先メールサーバ>　<from 送信元アドレス>　<to 送信先アドレス>
B0000055688＠xxxxx.xxx.xxx sent on [01/Mar/2008:10:29:51] to xxxx.xxx.jp from xxx＠xxxx.xxx.jp to xxxx＠xxxx.jp
