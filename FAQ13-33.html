Windows Server 2016 / 2019 / 2022 / 2025 のMSFCで構成したフェイルオーバークラスター環境でメールサーバは動作しますか？

E-Post Mail Server (x64) シリーズ、E-Post SMTP Server (x64) シリーズ製品は、Windows Server 2016 / 2019 / 2022 / 2025 に付属しているMSFCで構成したフェイルオーバークラスター環境での動作について、正式に対応しています。弊社では、MSFCで構成したフェイルオーバークラスター環境でメールサーバを構築するために、参照していただけるドキュメントを作成済みです。
