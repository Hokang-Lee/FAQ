SPA-PROで使用しているレジストリ
[COMMON]
HKEY_LOCAL_MACHINE
　->SOFTWARE
　　->EMWAC
　　　->IMS
　　　　->UserManager 　　(DWORD Default 1) 1:Windows Account, 0:SPA Account (SPA-Pro Only)
　　　　->Membership　　　(SZ Default NULL) NULL=Windows Accountの場合、
　　　　　マシンのローカルグループを使用/SPA Accountの場合ユーザアカウントＤＢ名
　　　　->SendDataTimeout (DWORD Default 60)sec (送信タイムアウト時間:default 60秒)
　　　　->AddressFamily 　　(DWORD Default 0) 0:IPv4 only, 1:IPv6 only, 2:IPv4&IPv6 mix
　　　　->HostName　　　　(SZ Default NULL) IPv6 host name.
　　　　->DomainNamesAreLocal (BINALY Default NULL) 運用ドメインリスト
　　　　->MailInCopy　　　　(BINARY Default NULL) Mail Carbon copy address.
　　　　->MailInMaxSize 　　(DWORD Default 0) 0:無制限 1メール当りの限界サイズ
　　　　->MailInBoxSize 　　(DWORD Default 0) 0:無制限 1メールボックス当りの保存限界サイズ
　　　　->RCPTMaximum 　　(DWORD Default 0) 0:無制限 あて先(RCPT TO:)数の制限
　　　　->MailInBoxDir　　　(SZ Default %HOME%\INETMAIL\INBOX) メールボックスフォルダ
　　　　->LastMsgId 　　　　(DWORD Default 0) メール受信カウンタ
　　　　->MailGroup 　　　　(SZ Default IMSUsers)
　　　　　 Windows Accountの場合、メールローカルグループ名
　　　　->MailSpoolDir　　　(SZ Default %SystemRoot%\SYSTEM32\SPA\MAIL) メール作業フォルダ
　　　　->LocalFailuresToPostmaster (DWORD Default 0)
　　　　　 0:Disable, 1:Enable エラーメールをpostmasterに送信
　　　　->PostMaster　　　　(SZ Default administrator) postmasterアカウント
　　　　->Aliases　　　　　　(KEY) エイリアス管理用
　　　　　->0 　　　　　　　(KEY) エイリアスその０
　　　　　　->Alias　　　　　(SZ Default NULL) エイリアス名
　　　　　　->MappedTo　　(SZ Default NULL) 実アドレス
　　　　　->1 　　　　　　　(KEY) エイリアスその１
　　　　　　: 以下繰り返し
　　　　->Lists 　　　　　　　(KEY) メーリングリスト管理用
　　　　　->[ML-list name1]　(KEY) メーリングリスト名１
　　　　　　->DeletesAttachment (DWORD Default 0)
　　　　　　　 添付ファイル取扱い方法 0:削除しない,1:削除する
　　　　　　->BounceAction　(DWORD Default 0)
　　　　　　　 未送信レポート先 0:送信しない,1:送信者に戻す,2:返信者を指定
　　　　　　->BouncesTo 　　(SZ Default NULL) 未送信レポート指定時メールアドレス
　　　　　　->ListPassword　(SZ Default NULL) 投稿パスワード
　　　　　　->ListSave　　　(DWORD Default 0) 投稿内容を保存 0:保存しない,1:保存する
　　　　　　->ListSaveFolder (SZ Default NULL) 投稿内容保存先
　　　　　　->MlId　　　　　(DWORD Default 0) カウンタ値
　　　　　　->MlToken 　　　(SZ Default NULL) 付加表題＆カウント方法
　　　　　　->Moderator 　　(SZ Default NULL) 管理人アドレス
　　　　　　->ModeratorControlJoin (DWORD Default 0) 入会要求は管理人へ転送
　　　　　　->ModeratorControlLeave (DWORD Default 0) 退会要求は管理人へ転送
　　　　　　->ReplyTo 　　　(DWORD Default 1) Reply-To:ヘッダを付加する 0:しない,1:する
　　　　　　->WhoCanSend (DWORD Default 1) 投稿者の条件 1:メンバーのみ,2:誰でも,3:管理人のみ
　　　　　　->Members 　　(KEY) 参加者管理用
　　　　　　　->[mail address] (KEY) 参加者メールアドレス
　　　　　　　　　: 繰り返し
　　　　　->[ML-list nameN] (メーリングリスト名１)
　　　　　　　: 以下繰り返し

HKEY_LOCAL_MACHINE
　->SOFTWARE
　　->SPA-PRO
　　　->IMS
　　　　->Domain
　　　　　->AcountNumber　　(KEY) ドメイン毎の登録者数上限値 (0=無制限)
　　　　　　->[domain name1] (DWORD Default 0)
　　　　　　　　:
　　　　　　->[domain nameN]
　　　　　->Folder　　　　　　(KEY) ドメイン毎のメールボックス保存フォルダ位置
　　　　　　->[domain name1] (SZ) eg.domain1.co.jp
　　　　　　　　:
　　　　　　->[domain nameN] (SZ) eg.domainN.co.jp
　　　　　->SmtpIP　　　　　(KEY)
　　　　　　 ドメインに対応するSMTP-IPアドレス（空白=マシンに設定されている全てのIPに応答）
　　　　　　->[domain name1] (SZ) eg.192.168.0.3
　　　　　　　　:
　　　　　　->[domain nameN] (SZ) eg.192.168.0.4
　　　　　->Pop3IP　　　　　(KEY)
　　　　　　 ドメインに対応するPOP3-IPアドレス（空白=マシンに設定されている全てのIPに応答）
　　　　　　->[domain name1] (SZ) eg.192.168.0.3
　　　　　　　　:
　　　　　　->[domain nameN] (SZ) eg.192.168.0.4
　　　　　->Imap4IP　　　　(KEY)
　　　　　　 ドメインに対応するIMAP4-IPアドレス（空白=マシンに設定されている全てのIPに応答）
　　　　　　->[domain name1] (SZ) eg.192.168.0.3
　　　　　　　　:
　　　　　　->[domain nameN] (SZ) eg.192.168.0.4
　　　　　->Operation　　　　(KEY) ドメイン毎の操作パスワード（空白=パスワード無し）
　　　　　　->[domain name1] (SZ) eg.password1
　　　　　　　　:
　　　　　　->[domain nameN] (SZ) eg.password2
　　　　->Secur
　　　　　->OpPassword 　　(DWORD Default 0=操作パスワードチェックしない）
　　　　　->SysPassword　　(SZ) メールサーバーの操作パスワード（空白=パスワード無し）
