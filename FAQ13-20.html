Web上から管理する機能はありますか？

E-Post Mail Server (x64) シリーズ、E-Post SMTP Server (x64) シリーズ製品には、Web管理マネージャのCGIプログラムが付属していますので、メールサーバにしたマシンに、IISなどのWebサーバソフトを導入・稼働させることで、Web上からメールサーバの設定を管理することが可能です。

Web管理マネージャのCGIプログラムを動作させるには、IISか、ApacheのいずれかのWebサーバソフトが必要ですが、PerlやPHPなどのスクリプトエンジンは一切不要で、単体で動作させることができます。

Web管理マネージャの画面の大部分は、HTMLで記述されていますので、必要な項目だけを残してHTMLでのページを編集し作り直すことで、自由なカスタマイズが可能です。
