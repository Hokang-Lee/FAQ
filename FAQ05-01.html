　　　　->LastMsgId　　　　　(DWORD Default 0) メール受信カウンタ
　　　　->MailGroup　　　　　(SZ Default IMSUsers) メールローカルグループ名

　　　　->LocalFailuresToPostmaster (DWORD Default 0)
　　　　　 0:Disable, 1:Enable エラーメールをpostmasterに送信

　　　　->Aliases　　　　　　　(KEY) エイリアス管理用
　　　　　->0　　　　　　　　　(KEY) エイリアスその０

　　　　　　: 以下繰り返し
　　　　->Lists　　　　　　　　(KEY) メーリングリスト管理用

　　　　　　->DeletesAttachment　(DWORD Default 0)
　　　　　　　 添付ファイル取扱い方法 0:削除しない,1:削除する
　　　　　　->BounceAction 　　　(DWORD Default 0)
　　　　　　　 未送信レポート先 0:送信しない,1:送信者に戻す,2:返信者を指定

　　　　　　->ListPassword　　　　(SZ Default NULL) 投稿パスワード
　　　　　　->ListSave　　　　　　(DWORD Default 0) 投稿内容を保存 0:保存しない,1:保存する

　　　　　　->MlId　　　　　　　　(DWORD Default 0) カウンタ値
　　　　　　->MlToken　　　　　　(SZ Default NULL) 付加表題＆カウント方法
　　　　　　->Moderator　　　　　(SZ Default NULL) 管理人アドレス
　　　　　　->ModeratorControlJoin (DWORD Default 0) 入会要求は管理人へ転送
　　　　　　->ModeratorControlLeave (DWORD Default 0) 退会要求は管理人へ転送
　　　　　　->ReplyTo 　　　　　　(DWORD Default 1)
　　　　　　　 Reply-To:ヘッダを付加する 0:しない,1:する
　　　　　　->WhoCanSend 　　　(DWORD Default 1)
　　　　　　　 投稿者の条件 1:メンバーのみ,2:誰でも,3:管理人のみ
　　　　　　->Members　　　　　　(KEY) 参加者管理用
